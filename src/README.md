# Dossier `dist/`

Ce dossier contiendra **votre fichier `index.html`**, que vous devez créer vous-même dans le cadre de l'exercice.

## Consignes

- Créez un fichier : ``dist/index.html``
- Ce fichier doit contenir la structure minimale d’une page HTML5 :
- `<!DOCTYPE html>`
- `<html lang="fr">`
- `<head>` avec `<meta charset="utf-8">` et `<title>…</title>`
- `<body>` contenant au moins un titre (`<h1>` ou `<h2>`) ou un paragraphe (`<p>`)
- Une fois votre `index.html` créé, **vous devez supprimer ce fichier `README.md`** si vous le souhaitez.
